# ImageManipulation
C# 图像基本操作，灰度化、二值化倾斜校正等

----------
在开发[PhoneDiscern](https://github.com/cfan1236/PhoneDiscern "PhoneDiscern") 项目时,使用Tesseract进行OCR识别,需要对图像做些基本处理，以便使识别的更准确。主要是自己需要处理图像做对比，是图像处理的一些实验性代码。

主要功能是灰度化、二值化倾斜校正等。
